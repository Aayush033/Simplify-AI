# 🚀 Simplify-AI: Fine-Tuned T5 Dialogue Summarizer Web App

An end-to-end web application utilizing a custom fine-tuned T5 model to perform abstractive dialogue summarization with a stateful session history tracking system.

---

### ⚙️ How to Run It Locally

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Aayush033/Simplify-AI.git


2. **Install Dependencies:**
  Ensure you have Python 3.10+ installed, then run:
  pip install -r requirements.txt

3. **Launch the Web Application Server:**
  uvicorn main:app --reload
